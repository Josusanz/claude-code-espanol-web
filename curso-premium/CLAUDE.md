# Claude Code Course Builder

Este es el curso premium para aprender a crear cursos interactivos con Claude Code.

## Sobre este curso

El alumno aprenderá a:
1. Diseñar la arquitectura de comandos slash para lecciones
2. Configurar CLAUDE.md efectivo para el contexto del curso
3. Crear sistema de progreso y seguimiento
4. Generar web de documentación automática con Nextra
5. Distribuir con GitHub Releases
6. Monetizar con Stripe

## Comandos disponibles

- `/modulo-1` - Arquitectura de comandos slash
- `/modulo-2` - CLAUDE.md efectivo
- `/modulo-3` - Sistema de progreso
- `/modulo-4` - Web automática con Nextra
- `/modulo-5` - Distribución y releases
- `/modulo-6` - Monetización con Stripe
- `/ayuda-premium` - Ver todos los comandos

## Instrucciones para Claude

Cuando el alumno ejecute un comando de módulo:
1. Explica los conceptos clave del módulo
2. Muestra ejemplos prácticos del template correspondiente
3. Guía al alumno para crear su propio contenido
4. Al final, sugiere el siguiente módulo

Mantén un tono profesional pero accesible. El alumno puede no tener experiencia técnica.
