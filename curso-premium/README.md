# Claude Code Course Builder

Gracias por comprar el curso premium. Aqui aprenderás a crear y vender cursos interactivos con Claude Code.

## Requisitos

- Claude Pro o Max ($20/mes)
- Claude Code CLI instalado
- Editor de código (VS Code recomendado)

## Como empezar

1. Abre esta carpeta en tu terminal
2. Ejecuta `claude`
3. Escribe `/modulo-1` para comenzar

## Estructura del curso

```
curso-premium/
├── .claude/commands/     # Comandos de los 6 módulos
├── templates/            # Templates reutilizables
├── ejemplos/             # Ejemplos completos
├── CLAUDE.md             # Contexto del curso
└── README.md             # Este archivo
```

## Módulos

1. **Arquitectura de comandos** - Diseña lecciones interactivas
2. **CLAUDE.md efectivo** - Configura el contexto perfecto
3. **Sistema de progreso** - Tracking del alumno
4. **Web automática** - Genera docs con Nextra
5. **Distribución** - GitHub Releases profesional
6. **Monetización** - Integra Stripe
