# Módulo 2: CLAUDE.md Efectivo

El archivo CLAUDE.md es el cerebro de tu curso. Define el contexto que Claude usa para responder.

## Por qué es importante

Sin CLAUDE.md, Claude no sabe:
- De qué trata tu curso
- Qué tono usar
- Qué comandos existen
- Cómo guiar al alumno

## Estructura recomendada

```markdown
# Nombre del Curso

Descripción breve del curso.

## Sobre este curso

Explicación de qué aprenderá el alumno.

## Comandos disponibles

Lista de todos los comandos con descripción.

## Instrucciones para Claude

Reglas específicas de comportamiento:
- Tono a usar
- Cómo manejar errores
- Qué hacer al final de cada lección
```

## Buenas prácticas

1. **Se específico** - No asumas que Claude sabe el contexto
2. **Define el tono** - Profesional, casual, técnico, etc.
3. **Lista comandos** - Claude necesita saber qué existe
4. **Incluye instrucciones** - Cómo responder en diferentes situaciones

## Ejercicio práctico

Revisa el archivo `templates/CLAUDE-template.md` y crea el CLAUDE.md para tu propio curso.

Preguntas a responder:
- ¿De qué trata tu curso?
- ¿Quién es tu alumno ideal?
- ¿Qué tono quieres usar?
- ¿Cuántos módulos/lecciones tendrás?
