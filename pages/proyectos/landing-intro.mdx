# Lección 2.1: Tu Primera Landing Page

En este proyecto vas a crear una **página web profesional** sin escribir código manualmente. Claude Code hará todo el trabajo técnico mientras tú te enfocas en el contenido.

## ¿Qué vamos a construir?

Una landing page completa con:

- **Hero section** - Título principal y llamada a la acción
- **Features** - 3-4 beneficios o características
- **Call to Action** - Botón para que el visitante tome acción
- **Footer** - Información de contacto y enlaces

<Callout type="info">
  Una landing page es una página web diseñada para convertir visitantes en clientes, suscriptores o leads. Es la base de cualquier presencia online.
</Callout>

## ¿Por qué una landing page?

| Sin landing page | Con landing page |
|------------------|------------------|
| Describes tu proyecto con palabras | Lo muestras visualmente |
| Das tu email o teléfono | Capturas datos de interesados |
| Dependes de redes sociales | Tienes tu propio espacio |
| Pareces amateur | Proyectas profesionalismo |
